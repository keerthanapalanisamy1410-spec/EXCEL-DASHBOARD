# 📊 Global Supermarket Sales Dashboard

## 📌 Project Overview

The Global Supermarket Sales Dashboard is an Excel-based business intelligence project designed to analyze sales performance, customer behavior, product profitability, and regional trends. The project utilizes Excel features such as VLOOKUP, Pivot Tables, Pivot Charts, Data Cleaning techniques, and Statistical Analysis to transform raw data into actionable insights.

---

## 🎯 Objective

- Analyze sales and profit performance across regions.
- Track key business KPIs and trends.
- Clean and prepare raw data for analysis.
- Identify top-performing products and customer segments.
- Create an interactive dashboard for business reporting.
